===============================
REALISTIC REGRESSION DATA
===============================

1. Realistic Dataset
--------------
Real-world datasets are usually
not perfectly clean.

Values often contain:
- variations
- irregular growth
- noise

--------------------------------

2. Previous Dataset vs Real Dataset
--------------
Previous datasets had
perfectly increasing values.

Example:

Experience   Salary

1            30000
2            40000
3            50000

This is usually unrealistic.

--------------------------------

3. Realistic Dataset Example
--------------
Experience   Salary

1            32000
2            41000
3            47000
4            65000
5            69000
6            85000

Notice:

Salary does not increase
by a fixed amount.

--------------------------------

4. Why Real Data Is Irregular
--------------
Real-world outcomes depend on
many factors.

Examples:
- skills
- company
- location
- negotiations
- market conditions

--------------------------------

5. Linear Regression On Real Data
--------------
Linear Regression does not require
perfect data.

It tries to learn the overall trend
from available observations.

--------------------------------

6. Important Understanding
--------------
Linear Regression does NOT memorize
training rows.

It learns a best fitting line
from the training data.

--------------------------------

7. Prediction Behaviour
--------------
Even if a training row exists:

Experience = 7
Salary = 92000

Linear Regression may predict:

92642.857

because it follows the learned line
instead of memorizing rows.

--------------------------------

8. Pattern Learning vs Memorization
--------------
Pattern Learning:
learn overall relationship.

Memorization:
store exact training examples.

Linear Regression focuses on
pattern learning.

--------------------------------

9. Real World ML Systems
--------------
Many production systems combine:

Known Data Lookup
+
Machine Learning Prediction

If exact data exists:
return stored answer.

Otherwise:
use ML prediction.

--------------------------------

10. Theory Reinforcement
--------------
Q: Why is this dataset more realistic
than previous datasets?

A:
Because real-world data usually contains
variations and irregular patterns.

--------------------------------

Q: Does real-world data always follow
perfect patterns?

A:
No.

Real-world data is often noisy
and irregular.

--------------------------------

Q: Why can Linear Regression still work
when data is not perfect?

A:
Because it tries to learn
the overall trend in the data.

--------------------------------

11. Practical Reinforcement
--------------
Q: What happened after adding:

Experience = 7
Salary = 92000

to the dataset?

A:
The prediction changed because
the model learned from the updated data.

--------------------------------

Q: Did the model memorize 92000?

A:
No.

The model still followed
the best fitting line.

--------------------------------

12. Important Learning Principle
--------------
A model can learn useful patterns
even when data is not perfectly clean.

Real-world datasets are rarely perfect.
