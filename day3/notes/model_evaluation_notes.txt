===============================
MODEL EVALUATION
===============================

1. Why Model Evaluation Needed?
--------------
Model evaluation is used to check
how well model predictions
match actual answers.

--------------------------------

2. Important Understanding
--------------
Building model alone is NOT enough.

We must measure:
how good predictions are.

--------------------------------

3. Prediction vs Actual Answer
--------------
Model predicts outputs.

These predictions are compared
against actual correct answers.

--------------------------------

4. Mean Absolute Error (MAE)
--------------
Mean Absolute Error (MAE)
calculates average prediction error
between:
- actual outputs
- predicted outputs

--------------------------------

5. Simple MAE Understanding
--------------
MAE means:
average prediction mistake.

--------------------------------

6. Smaller MAE
--------------
Smaller MAE means:
model predictions are closer
to actual answers.

Usually indicates:
better model performance.

--------------------------------

7. Larger MAE
--------------
Larger MAE means:
predictions are farther
from actual answers.

Usually indicates:
weaker model performance.

--------------------------------

8. Important Evaluation Flow
--------------
Prediction
        ↓
Compare with Actual Answers
        ↓
Calculate Error
        ↓
Measure Model Quality

--------------------------------

9. Why Prediction Done on Testing Data?
--------------
Testing data is unseen during training.

Prediction on testing data helps check:
how well model performs
on unseen data.

--------------------------------

10. Important ML Goal
--------------
Good ML models should:
generalize well
on unseen data.

Not simply memorize training examples.

--------------------------------

11. mean_absolute_error()
--------------
mean_absolute_error()
compares:
- actual outputs
- predicted outputs

Then calculates:
average prediction error.

--------------------------------

12. Prediction vs Evaluation
--------------
Prediction:
generating outputs
for given input data.

Evaluation:
measuring how accurate
those predictions are.

--------------------------------

13. Important ML Pipeline
--------------
Data
 ↓
Train/Test Split
 ↓
Create Model
 ↓
fit() → Learning
 ↓
predict() → Predictions
 ↓
Compare Predictions vs Actual Answers
 ↓
Evaluation Metrics

--------------------------------
